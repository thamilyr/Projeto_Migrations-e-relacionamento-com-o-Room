## 📌 Sobre o projeto

Este projeto foi desenvolvido com o objetivo de aprofundar conhecimentos em persistência de dados no Android utilizando o **Room Database**.

A aplicação demonstra como lidar com:

* criação de banco de dados local
* versionamento de banco com migrations
* relacionamentos entre tabelas
* organização de dados em camadas

---

## 🧠 Conceitos aplicados

* Room Database (Android Jetpack)
* Migrations (versionamento de banco)
* Relacionamentos entre entidades (1:N, N:N)
* DAO (Data Access Object)
* Arquitetura em camadas

---

## 🛠️ Tecnologias utilizadas

* Kotlin
* Android Studio
* Room Persistence Library
* SQLite

---

## 📂 Estrutura do projeto

* **Entity** → Representação das tabelas
* **DAO** → Acesso aos dados
* **Database** → Configuração do Room
* **Migrations** → Evolução do banco de dados
* **Repository** → Camada de abstração

---

## 🚀 Funcionalidades

✔ Criação e gerenciamento de banco local
✔ Inserção e leitura de dados
✔ Atualização de estrutura com migrations
✔ Relacionamento entre entidades
✔ Persistência eficiente e estruturada
